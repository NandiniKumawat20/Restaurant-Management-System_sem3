## Project Description

The **Restaurant Management System** is a web-based application designed to automate and manage restaurant operations efficiently.  
The system helps in managing menu items, customer orders, billing, and user roles such as admin and staff.  
It improves order accuracy, reduces manual work, and enhances overall restaurant productivity.
